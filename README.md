# easy-frame

### 项目介绍
easy-frame基于springboot2，整合springboot+shiro+mybatis-plus+beetl+swagger+redis。内置权限、部门、参数、字典、代码生成等模块。暂时只支持MySQL，后续会添加Oracle等数据库支持。目前处于完善阶段...
